## 🧠 Overview
**GSmartControl** is a specialized Windows tool designed to monitor and analyze hard drive and SSD health. It reads S.M.A.R.T. attributes, evaluates performance, and identifies early warning signs of potential failures, giving users time to safeguard their data.  

With a clean and accessible interface, the application is suitable for newcomers while also offering advanced diagnostics for experienced users. GSmartControl enables monitoring of disk performance, running self-tests, and tracking trends over time to maintain reliable and secure storage.

---

## 🚀 Key Features

| Feature | Description |
|----------|-------------|
| ⚡ **S.M.A.R.T. Analysis** | Provides detailed health and status reports for each drive. |
| 🎨 **Comprehensive Self-Tests** | Run short, extended, and conveyance tests to check disk integrity. |
| 🔄 **Early Warning Alerts** | Notifies users when a disk shows potential issues. |
| 📐 **Supports HDD & SSD** | Compatible with mechanical hard drives and solid-state drives. |
| 💾 **Visual Health Overview** | Offers a clear graphical display of drive status and performance. |
| 🌐 **Performance Tracking** | Monitor disk metrics over time to detect gradual decline. |
| ⚙️ **Lightweight & Efficient** | Minimal impact on system performance while running diagnostics. |
| 🔒 **Reliable & Accurate** | Trusted tool worldwide for maintaining drive health and stability. |
